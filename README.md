# Complejidad_Algoritmica_OptiRed
El objetivo de este proyecto es desarrollar un sistema capaz de analizar las distancias entre múltiples antenas y determinar su disposición más óptima.

## Datos Generales:
**EMPRESA_OPERADORA**
- 1 = América Móvil Perú S.A.C.
- 2 = Telefónica del Perú S.A.A.
- 3 = Viettel Perú S.A.C.
- 4 = Entel Perú S.A.
